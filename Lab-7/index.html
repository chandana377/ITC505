This Mad Lib project was built using ExpressJS for backend routing and modern JavaScript (fetch API) for frontend dynamic updates. The goal was to create a fun, interactive story generator that processes input without reloading the page.

The backend validates form inputs and sends a JSON response containing the completed story. The frontend then displays the story beneath the form.

CSS provides a light blue background, dark purple ghoulish font for the heading, and a clean, centered layout.
